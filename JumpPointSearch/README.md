# 사용방법

<br>

- 왼쪽 클릭 : 출발지 설정

- 오른쪽 클릭 : 목적지 설정

- Shift + 왼쪽 클릭 : 장애물 설치

- Ctrl + 왼쪽 클릭 : 장애물 제거

- Enter : 탐색 시작

- Space : 초기화
